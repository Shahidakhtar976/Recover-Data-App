# Recover-Data-App
First screen is splash screen 
Click on the splash screen Then you will go to the login/sign up screen
If user already registered then click on login
If user not registered then click on sign up
After the login/sign up you will go to home page
In home page you have four buttons Images,Videos,About Us and Help
If u want to recover images/videos just tap on images/videos
And then scan yours images/videos
